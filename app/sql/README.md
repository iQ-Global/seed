# SQL Files

This directory is for **your application's SQL files**.

## Usage

Store your database schemas, migrations, and seed data here:

- `schema.sql` - Your database schema
- `seed-data.sql` - Initial data for your app
- `migrations/` - Your migration files

## System SQL Files

System module SQL files are located in their respective module directories:
- Database sessions: `system/modules/session/sessions.sql`
